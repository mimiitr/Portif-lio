Projetos de Destaque

Nessa página, estão listados os principais projetos acadêmicos realizados por mim durante meu aprendizado. Cada projeto trata de uma competência diferente, que aprendi durante meus estudos em sala de aula.

Para mais detalhrd, em cada seção, há um botão que leva ao devido repositório no Github.

Este projeto consiste na prática do aprendizado da utilização de Bootstrap no CSS. Realizado no ano de 2024, foi um dos meus primeiros projetos utilizando um framework, o que serviu como aprendizado e aprimoramento das minhas habilidades com CSS.

"Glorious Hair" foi um projeto proposto em sala de aula também no ano de 2024. O objetivo era criar uma interface com base nos conhecimentos adquiridos naquele ano. Com ajuda do meu grupo de trabalho, conseguimos desenvolvê-lo completamente.

Katchau Postos

Com a temática do filme "Carros", da Pixar, "Katchau Postos" foi uma das minhas primeiras aplicações práticas de Javascript na Web, utilizando de troca de imagens para exibir o resultado de cálculos. Este metódo foi também aplicado no Jogo da Sequência, neste site.
